**ขั้นตอนการทำงาน**

1. Assign งานให้แต่ละคน โดยแต่ละคนมีหน้าที่ดังนี้
- **ภควรรธ**: รับหน้าที่ในส่วนของการทำการรับค่า Input
- **ศิวภาส**: รับหน้าที่ในส่วนของ Quick และ Bubble Sort
- **รัชภูมิ**: รับหน้าที่ในส่วนของการทำการรับค่า Output และในส่วนของ main.py

2. แต่ละคนทำการสร้าง branch แยกของตนเอง
- **ภควรรธ**: Phakawat_2275
- **ศิวภาส**: Siwapad_0265
- **รัชภูมิ**: Ratchapoom_2314

3. แต่ละคนทำการเขียนโค้ดในส่วนของที่แต่ละคนได้รับมอบหมาย
4. เมื่อทำโค้ดที่ได้รับมอบหมายเสร็จแล้วทำการ push ขึ้น github ใน branch ของตนเอง
5. ทำการตรวจสอบความถูกต้องโค้ดของแต่ละคนว
6. ทำการ pull request แล้วไป Merge รวมกันที่ Main
